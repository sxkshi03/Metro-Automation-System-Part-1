# Metro Automation System (Part - 1)

## Objective
To develop a PLC-based metro automation system that controls train movement through three blocks of train section using sequential logic, timers, and signal interlocking.

## Features

-Sequential train movement  
-One-shot start trigger  
-Timer-based block travel  
-Block interlocking  
-Signal indication  
-Break-before-make transition  

## PLC Instructions Used

- ONS
- TON
- OTL / OTU
- RES
- Compare Logic
